# Shrimp Tail Menu

A menu project for shrimp tail restaurant.

## Getting Started

Clone this repo and run locally:

```sh
bun install
bun run dev
```

## Deployment

Build for production:

```sh
bun run build
```

Deploy the contents of the `dist/` folder to GitHub Pages or your preferred static host.
